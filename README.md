# Parin Shah Portfolio

Static portfolio site for Parin Shah.

## Local Preview

Open `index.html` directly in a browser, or serve the folder with any static server.

For a dependency-free preview in PowerShell:

```powershell
.\preview.ps1
```

Then open `http://localhost:4173`.

## Deploy

This folder is ready to deploy as a static site.

### Vercel

1. Create a new project from this `portfolio-site` folder.
2. Framework preset: `Other`.
3. Output directory: leave empty.
4. Deploy.

### Netlify

1. Drag and drop this `portfolio-site` folder into Netlify Drop, or connect the repo/folder.
2. Publish directory: `portfolio-site` if deploying from repo root.
3. Deploy.
